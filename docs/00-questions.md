## 案B選定経緯（決定済み）

### 2026-04-13: プラットフォームを Webflow → WordPress + SWELL に変更

**背景**: 初期はWebflow前提で構築を進めていたが、案件規模・コスト・
KARINTOW担当者の既存スキル・ロックインリスクを再評価した結果、
WordPress + SWELL + ConoHa WING 構成が最適と判断。

**決定事項**:
- ホスティング: **ConoHa WING ベーシック**（¥941/月）
- テーマ: **SWELL**（¥17,600 買切）
- CMS: **WordPress 6.x**（標準 + ACF Pro + Custom Post Type UI）
- Webflowサイト（ID: `69d1f22c1d19cda39fee4fec`）は**廃止**

**主な判断理由**:
- 月額コストが現状より安い（¥1,738 → ¥941）
- KARINTOW既存のWP運用経験を活用
- ロックイン回避（WP標準）
- 管理型ホスティングでroot権限喪失事故が構造的に起きない
- 旧WP→新WPでデータ移行が比較的スムーズ

この決定に伴い、v1 で記載されていたWebflow関連の質問は
**CLOSED** または **OBSOLETE** として除外した。

### Q02: ブランドカラー

現行旧サイトCSSに実在するのはピンク `#e48898` のみ。メモv2記載の `#CC2E32` は
旧サイトに存在しない色。判断肢：

- **案A**: 赤 `#d4002a` — 純赤、中年男性向け、本格感
- **案B**: 赤 `#b91c1c` — 深紅、シック、高級感
- **案C**: ピンク `#e48898` — 旧サイトを継承、既存ファンに優しい

**推奨**: 案A（ターゲット層に最も適合）

### Q04: 旧URL保全（部分解決）

残る検討事項は `/one/{ID}/` のみ → `.htaccess` 301リダイレクトで対応予定。
