## このドキュメントについて

株式会社KARINTOWが運営する撮影会スタジオ「Gスタ撮影会」（studio-g.net）の
HP刷新プロジェクトにおける、システム全体の設計仕様書です。

現行WordPressサイト（CentOS 7 + WP 4.9.26）からWebflowベースの新サイトへの
移行を前提に、サイト構成・データモデル・運用フロー・インフラ構成を定義します。
